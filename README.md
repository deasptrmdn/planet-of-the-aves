# planet-of-the-aves

Planet of The Aves is an immersive educational mobile application designed to help users learn about bird habitats, adaptations, and reproduction lifecycles through interactive exploration.

Project Features
Environment Exploration
Bird Information
Interactive 3D Bird Exploration
Lifecycle Learning
Educational Quiz
Design Prototype
